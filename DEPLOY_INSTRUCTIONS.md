# Инструкция по развертыванию на Render (Free Tier)

У вас уже настроен проект для автоматического развертывания через Render Blueprints. Это самый простой способ.

### Шаг 1: Обновите код на GitHub
1. Запустите файл `update_github.bat` в папке проекта.
2. Дождитесь сообщения "Done!".

### Шаг 2: Создайте сервис на Render
1. Зарегистрируйтесь или войдите на [dashboard.render.com](https://dashboard.render.com/).
2. Нажмите кнопку **"New +"** вверху справа и выберите **"Blueprint"**.
3. Подключите ваш GitHub аккаунт, если еще не сделали этого.
4. Найдите ваш репозиторий `stitch` в списке и нажмите **"Connect"**.
5. Render автоматически обнаружит файл `render.yaml`.
6. Нажмите **"Apply"** внизу страницы.

### Шаг 3: Готово!
Render сам сделает следующее:
1. Создаст базу данных PostgreSQL (бесплатный тариф).
2. Создаст веб-сервис Python.
3. Свяжет их друг с другом.
4. Запустит установку зависимостей и старт приложения.

Процесс займет около 2-5 минут. В конце вы увидите URL вашего сайта (например, `stitch-messenger.onrender.com`).

---

### Важные моменты Free Tier:
* **Спящий режим**: Если сайтом не пользоваться 15 минут, он "уснет". Первое открытие после сна займет около 30-50 секунд.
* **База данных**: Бесплатная база данных удаляется через 90 дней (но ее можно продлить или пересоздать).
